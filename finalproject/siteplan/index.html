Video Game Library – Site Plan

WDD 231 Individual Project

Site Name

Video Game Library

This name was selected because the website focuses on providing a collection of video game information in a structured and easy-to-browse library format. The name clearly represents the main purpose of the site and is easy to recognize and remember.

Optional domain: videogamelibrary.org

Site Purpose

The purpose of this website is to provide an interactive and user-friendly platform where visitors can explore a dynamic catalog of video games. The site will display information such as game title, genre, platform, release year, and descriptions using data loaded from a JSON file. Users will be able to view detailed information and suggest new games through a form.

Scenarios

  • What video games are available for a specific platform such as PC or PlayStation?
  • Where can I find detailed information about a game’s genre and release year?
  • How can I suggest a new video game to be added to the library?

Color Scheme Legend

Dark Blue (#1e293b) - Headers & Navigation
Light Gray (#f1f5f9) - Backgrounds
Accent Green (#22c55e) - Buttons & Highlights

Typography

  • Orbitron: Used for headings and main titles to give a modern and gaming-inspired look.
  • Roboto: Used for body text and paragraphs for readability and accessibility.
